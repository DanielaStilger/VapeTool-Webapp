# Vape Tool Webapp

![Demo](docs/cloud.jpg) This project uses pnpm, Vite, antd, Flowbite-react (Tailwindcss), standardjs and firebase.

## Install

```bash
pnpm install
```

## Develop

```bash
pnpm dev
```

## Linter & Formatter
We use only [JavaScript Standard Style](https://standardjs.com)

```bash
pnpm lint
